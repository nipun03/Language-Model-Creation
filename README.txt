Extract the zip file in a folder say "C:\xyz" as path

To the run the file:
1. Open command prompt
2. Make sure your path is assigned to xyz folder. For that "cd C:\xyz"
3. Type "py gupta_assignment2.py" to execute the python file.
4. If you want you can save the output in .txt file(or any other file of your choice) by "py gupta_assignment2.py > output.txt" to check the probabilities in unigram, bigram and trigram model, and accuracies of each model.
